## Port Scanner

Port Scanner built with Python that scans a host for open TCP ports using the socket library.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

## How It Works
Creates a TCP socket for each port in the given range and attempts a connection.
Returns a list of open ports found on the target host.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

## How To Use
```
scan("scanme.nmap.org", 20, 100)
```

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

## Skills Demonstrated
- Python socket programming
- TCP/IP networking fundamentals
- Basic cybersecurity tooling
